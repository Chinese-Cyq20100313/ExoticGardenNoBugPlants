# ExoticGarden 异域花园

ExoticGarden 是一个 [Slimefun 4](https://github.com/TheBusyBiscuit/Slimefun4) 附属插件，向 Slimefun 添加了各种植物、水果、蔬菜与食物。

你可以在我们**过期的**BukkitDev页面找到更多信息:
https://dev.bukkit.org/projects/exotic-garden
